Adição

Use a função sum() para somar números.

Essa função tem como parâmetro uma lista, onde todos os seus valores serão somados.

O resultado retornado deve ser 10

print (sum( [1, 2, 3, 4] ) )

>>> 10

Subtração

Use a função sub() para subtrair números.

Assim como a função de adição, essa função tem como parâmetro uma lista, onde todos os valores serão subtraídos

O resultado retornado deve ser 4

print (sub ( [10, 3, 3] ) )

>>> 4

Multiplicação

use a função mult() para subtrair números.

Essa função também possui uma lista como parâmetro, na qual todos os seus valores serão multiplicados

O resultador deve ser 20

print (mult ( [10, 2] ) )

>>> 20

Divisão

Use a função div() para subtrair números

Essa função recebe lista como parâmetro, onde o primeiro valor será dividido pelos outros

O resultado deve ser 5

print( div ( [20, 2, 2] ) )

>>> 5

Exponenciação

use a função expo() para realizar uma exponenciação

Essa função recebe uma lista, onde o valores são exponenciados

O resultado deve ser 4096

print( expo ( [4, 2, 3] ) )

>>> 4096

Radiciação

Use a função sqrt() para realizar uma radiciação

Diferente das outras, essa função tem dois inteiros como parâmetro

O primeiro valor deve ser o número e o segundo deve ser a raiz

Os resultados retornados devem ser 30 e 8

print( sqrt (100, 3) )

>>> 30

print( sqrt (81, 2) )

>>> 9

Grau para Radianos

Use a função radian() para realizar a converção

Essa função deve ter um número inteiro como parâmetro

O resultado retornado deve ser 0.17453292519943295

print ( radian (10) )

>>> 0.17453292519943295

Celsius para fahrenheit

Use a função fah() para converter Celsius para fahrenheit

Essa função tem como parâmetro um número inteiro

O resultado retornado no código abaixo deve ser 86.0

print( fah (30) )

>>> 86.0

fahrenheit para Celsius

Use a função cel() para converter fahrenheit para Celsius

Essa função tem como parâmetro um número inteiro

O resultado retornado no código abaixo deve ser 30

print( cel (86) )

>>> 30.0
